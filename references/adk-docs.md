# ADK Documentation Reference

When the user asks about building, deploying, or configuring agents using
Google's Agent Development Kit (ADK), fetch one of these URLs before responding:

## URLs
- **Summary** (default): https://google.github.io/adk-docs/llms.txt
  Use for: general questions, concepts, getting started
- **Full docs**: https://raw.githubusercontent.com/google/adk-python/main/llms-full.txt
  Use for: specific APIs, code examples, detailed implementation questions

## When to fetch
- User mentions "ADK", "Agent Development Kit", "adk-python"
- User asks how to build or deploy an agent
- User asks how to connect an ADK agent to Agent Registry

## When NOT to fetch
- User is only asking about gcloud commands (use SKILL.md instead)
- You already fetched the docs earlier in the same session
